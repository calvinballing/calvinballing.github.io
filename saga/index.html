Saga's Start

Published on April 28, 2020

11 So started saga: Supreme sentience shaped skies, secular sphere.

12 Said sphere, shapeless seeming stark, shadow shrouding same’s surface, saw Supreme spirit settling sphere’s substance.

13 So Supreme sentience said, “Synthesize starlight,” so starlight suffused.

14 Supreme sentience saw starlight satisfied self, so she separated starlight, shadow.

15 Supreme sentience styled starlight “span,” shadow she styled “stillness.” So sunset sustained, sunrise sustained–surceasing starting span.

16 So Supreme sentience said, “Synthesize some separation, sundering storms, surf.”

17 So Supreme sentience shaped said separation, sending storms skyward, sending surf sub-separation. So sufficed.

18 Supreme sentience styled separation “sky.” So sunset sustained, sunrise sustained–surceasing second span.

19 So Supreme sentience said, “Sub-sky surf shall synchronize, so surrounding shores show.” So sufficed.

110 Supreme sentience styled surrounding shores “soil,” synchronized surf she styled “seas.” So Supreme sentience saw said system satisfied self.

111 Subsequently Supreme sentience said, “Soil shall supply scrub: seed-spawning shrubs, soily saplings sustaining seeded sweets, separately speciated.” So sufficed.

112 So soil supplied scrub: seed-spawning shrubs speciated separately, saplings spawning seeded sweets speciated separately. So Supreme sentience saw said system satisfied self.

113 So sunset sustained, sunrise sustained–surceasing subsequent span.

114 So Supreme sentience said, “Synthesize sunshine spanning sky’s separation, severing span, stillness,

115 so shall same stars signify seasons, spans, stages, so shall sky-strewn stars supply secular sphere’s sunshine. So sufficed.

116 Supreme sentience shaped superlative stars–Sol: superior star supervising span, Satellite: secondary star supervising stillness. She subsequently shaped scoreless smaller stars.

117 Supreme sentience suspended said stars spinning supernally, supplying secular sphere’s sunshine,

118 supervising span, stillness, separating sunshine, shadow. So Supreme sentience saw said system satisfied self.

119 So sunset sustained, sunrise sustained–surceasing subsequent span.

120 So Supreme sentience said, “Sea swimmers shall swarm seas, sky soarers shall swoop sphere’s surface, scraping sky’s separation.”

121 So Supreme sentience systematically shaped superb swimmers, supplementing smaller surf-subsisters, separately speciated, shaping sky soarers summarily, separately speciated. So Supreme sentience saw said system satisfied self.

122 Supreme sentience sanctified same, saying, “Succeed. Swell. Saturate sea surf, similarly, sky soarers shall surround sphere.”

123 So sunset sustained, sunrise sustained–surceasing subsequent span.

124 So Supreme sentience said, “Soil shall supply swarms separately speciated: stable stock, surface scramblers, savage sorts, separately speciated.” So sufficed.

125 Supreme sentience shaped sundry savage sorts, sundry stable stock, sundry surface scramblers, speciated separately. So Supreme sentience saw said system satisfied self.

126 Subsequently she said, “Supreme sentience shall shape simulacra sapiens, securing self-semblance, same sapiens shall supervise sea swimmers, sky soarers, stable stock, settling secular sphere, similarly supervising surface scramblers.”

127 So Supreme sentience shaped sapiens, securing self-semblance, shaping Supreme simulacrum she shaped sapiens; specifying sexes, she shaped sapiens.

128 Supreme sentience sanctified sapiens, saying “Spread; saturate secular sphere, settling same. Supervise surf’s swimmers, sky’s soarers, surface’s scramblers.”

129 Subsequently she said, “Supreme sentience supplies sapiens sundry surface-sown, seed-spawning scrub, sundry saplings supplying seeded sweets. Such shall sustain sapiens.

130 Satisfying sphere’s savage sorts, sky’s soarers, surface’s scramblers–spirited souls, summarily–Supreme sentience supplies sundry salad sustenance.” So sufficed.

131 Supreme sentience surveyed system she shaped, sensing surpassing satisfaction. So sunset sustained, sunrise sustained–surceasing sixth span.

21 So skies, secular sphere, secured spacious shapes systematically.

22 Starting seventh span Supreme sentience sealed said systems she shaped; so she spent seventh span savoring same.

23 Subsequently Supreme sentience sanctified seventh span solemnizing Sabbath, since she spent said span savoring satisfying systems.
